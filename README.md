**Universe** is a demo social media platform built for students — a place to connect, share posts, and explore a campus-style feed experience.

This repository contains:
- **Frontend (Web)**: React
- **Backend (API)**: Express.js (Node.js)
- **Mobile App**: React Native

> Demo project.

---

## Features

- Student-style social feed (posts, likes, comments, follows)
- User authentication with valid academic email
- Real time chatting ( direct and groups )
- Profiles
- Responsive web UI
- Mobile app companion built with React Native

---

## Tech Stack

- **Web**: React
- **API**: Node.js + Express.js
- **Mobile**: React Native
